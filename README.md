# Gerenciador de Senhas

Este projeto é um **Gerenciador de Senhas** que permite ao usuário adicionar, alterar, excluir e visualizar senhas de forma segura. O objetivo é proporcionar uma maneira simples e prática de gerenciar senhas de diversos serviços.

## Funcionalidades

- **Adicionar Senhas**: Permite ao usuário adicionar novas senhas ao gerenciador.
- **Alterar Senhas**: Possibilita a edição de senhas existentes.
- **Excluir Senhas**: Remove senhas que o usuário não deseja mais armazenar.
- **Visualizar Senhas**: Exibe uma lista das senhas salvas para fácil consulta.

## Estrutura do Projeto

- **src/**: Diretório principal contendo o código da aplicação.
  - **components/**: Componentes de interface reutilizáveis (formulários, listas, etc.).
  - **screens/**: Telas da aplicação, como a tela principal de exibição de senhas e a tela de edição.
  - **utils/**: Funções auxiliares, como criptografia e manipulação de dados.

## Tecnologias Utilizadas
# até o momento
- **React Native / Expo**: Para o desenvolvimento do front-end.
- **JavaScript / TypeScript**: Linguagem principal do projeto.


## Instalação e Configuração

1. **Clone o repositório**:

    ```bash
    git clone <URL_DO_REPOSITORIO>
    cd <NOME_DO_REPOSITORIO>
    ```

2. **Instale as dependências**:

    ```bash
    npm install
    ```

3. **Inicie o aplicativo**:

    ```bash
    npx expo start e escolhar "a"=android e "w" web
    ```

## Como Usar

1. **Adicionar uma nova senha**: No menu principal, clique em "Adicionar Senha", insira as informações do serviço e a senha e salve.
2. **Editar uma senha**: Selecione uma senha na lista para editar os detalhes e atualizá-la.
3. **Excluir uma senha**: Na lista de senhas, selecione uma senha e clique em "Excluir" para removê-la.
